​​Iniciando nossa playlist do projeto Vakinha Burger onde vamos construir um app completo de delivery com pagamento por PIX.

Será construído um aplicativo em Flutter com um backend feito em Dart com Shelf e integrando a api de PIX da Gerencianet. 

Pré Requisitos para o projeto abordados na aula: 

- Abrir uma conta na Gerencianet (https://gerencianet.com.br/) 

- Criar uma conta na AWS 

- Criar uma conta no https://www.cloudns.net/
